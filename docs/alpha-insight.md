## 用途

Alpha洞察把热点新闻和研究任务交给 AI consumer 处理，完成后回写 Markdown 研报，并通过通知渠道提醒你查看。

## 常用操作

- 创建任务前先确认通知渠道可用。
- 任务提交后关注状态变化：创建中、执行中、完成或失败。
- 完成后的 Markdown 内容可作为组合树文章沉淀。
