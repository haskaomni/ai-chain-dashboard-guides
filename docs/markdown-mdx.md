## 用途

文档节点让股票研究不只停留在列表中。你可以把交易计划、基本面摘要、AI 研报和图表复盘沉淀到对应主题下。

## 常用操作

- 点击文档节点打开阅读页。
- 右键文档节点进入编辑弹窗。
- Markdown 支持常规文本格式；Safe MDX 支持受控组件展示。
- 只读订阅组合中的文章不能直接编辑。
