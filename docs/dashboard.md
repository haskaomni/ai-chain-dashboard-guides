## 界面结构

- 左侧是组合树，集中管理文件夹、股票、文章和数据视图。
- 中间是主工作区，可在「发现」「持仓」「图表」「期权」以及打开的节点页之间切换。
- 右侧是公司概况，选中股票后展示报价、基本面和机构预测。
- 底部指数栏显示主要市场指数，帮助判断大盘环境。

## 常用操作

- 点击顶部标签在发现页、持仓页、图表页和已打开内容之间切换。
- 使用中间顶部搜索框快速跳转股票。
- 使用左右边缘的折叠按钮扩大图表、持仓或发现页阅读空间。
- 右下角工具区可打开 API 面板、帮助文档、Telegram Bot 绑定和退出登录。
