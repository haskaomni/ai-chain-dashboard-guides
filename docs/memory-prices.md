## 用途

内存价格用于观察存储和 AI 服务器供应链变化，适合与半导体、服务器、云厂商股票一起分析。

## 阅读方式

- 关注价格趋势而不是单次报价。
- 对比不同品类或规格的价格变化。
- 与「AI瓶颈」和图表页联动判断产业拐点。
